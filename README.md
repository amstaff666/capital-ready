# CapitalReady
Premium kliendiportaal ettevõtte laenuvalmiduse juhtimiseks.
## Põhiloogika
- klient loob konto
- klient logib sisse
- klient täidab ettevõtte/laenuandmed
- klient läbib 6 põhipanga kontrolli
- klient näeb dashboardis, mis on puudu
- haldur näeb hiljem sama case'i halduri töövoos
## 6 kohustuslikku panka
- Swedbank
- SEB
- LHV
- Coop
- Luminor
- Citadele
